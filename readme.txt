== Description ==

Fare calculator can manage the fare amount between the distance points A and B. For eg: if a user wants to travel from one point to another point, he/she needs a fare amount about their distance travel, this plugin will be helpful and it is responsive too,
In Admin panel goto fare settings on admin menu, then enter the lat and long for the map area to display
and enter the service and its price to calculate, After finishing save the data, the data which saved can be preview below table on fare settings.
--Enjoy--
Copy the shorcode into pages or posts: [fc_fare]

== Installation ==

    Upload the entire fare calculator folder to the /wp-content/plugins/ directory.
    Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'fare calculator' menu in your WordPress admin panel.

For basic usage, you can also have a look at the plugin homepage.
